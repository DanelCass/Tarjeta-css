Pinza Electrica

Una pinza eléctrica, también conocida como pinza de electricista o alicates de electricista, es una herramienta manual indispensable para trabajos de electricidad. Está diseñada para cortar, doblar, pelar y sujetar cables y alambres.

Martillo

Un martillo es una herramienta básica que consta de dos partes principales: una cabeza pesada y un mango o asa. La cabeza, generalmente de metal, se utiliza para golpear, clavar, o dar forma a objetos, mientras que el mango proporciona un agarre y apalancamiento.

Ajutable

Una llave ajustable, también conocida como llave inglesa o llave de tuercas, es una herramienta manual versátil que se utiliza para apretar o aflojar tuercas y tornillos de diferentes tamaños.
